Hexo | simple start

학습한 내용을 정리할 블로그가 하나 필요했는데 이것 저것 알아보다가 node.js를 기반으로 하는 블로그 프레임 워크인 hexo를 발견하게 되었습니다. hexo로 결정한 이유는 github pages를 활용해서 비교적 간단하게 정적 사이트를 만들 수 있고, node.js를 활용하게 되어 웹기술에 대한 학습도 된다는 장점이 있습니다. 게다가 Markdown을 지원하기 때문에 글을 편하게 작성할 수 있습니다.

Simple Start

Install

node.js와 git이 설치되어 있어야 합니다.

터미널에서 npm(node.js package manager)으로 hexo-cli를 설치 합니다.

다음으로 프로젝트 폴더를 생성함과 동시에 초기화를 합니다.

프로젝트의 각종 모듈을 관리하기 위해서 npm을 설치해야 합니다.

  • 참고 : Setup

Start the server

새롭게 생성된 사이트를 local server를 통해서 웹브라우저(http://localhost:4000/)에서 확인해볼 수 있습니다.

Post

새로운 글을 작성 합니다. (/source/_posts/ 생성)

정적 파일을 생성합니다.

생성된 정적 파일을 deloy 합니다.

이것 저것 만지다보니 프레임워크의 구조를 잘 파악해야 자유롭게 활용할 수 있을 것 같습니다. 이리 저리 검색해봤는데 자료(영어자료 포함)가 많지 않을 것 같습니다. 삽질이 예상됩다.
